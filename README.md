## Running Locally

### Clone the Repository

```bash
git clone https://github.com/tp-shivha-shakthiy/Portfolio.git
cd Portfolio
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
